# four-in-a-row-game (C++)

A work-in-progress C++ implementation of the Four-in-a-Row game.

## Current Features
- Game loop
- Board representation
- Win condition checking (horizontal, vertical, diagonal)
- Unique player chip color

## Planned Improvements
- User interface / visualization
- Input validation
- Code cleanup and refactoring

This project is being developed to practice game logic, control flow,
and algorithmic thinking in C++.
